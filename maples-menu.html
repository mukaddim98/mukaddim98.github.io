Maples Menu

A tablet app for ordering daily meals

Project Lifespan: Feb-April 2021; Project State: Completed; Project Type: Co-Op project

Overview: It’s a Co-Op project from a company called ImmediaC. The idea is to build a tablet-based app that will be used by a waiter to make orders for residents in a retirement home. The app displays a daily menu along with meals and their items. It tracks resident’s dietary information and makes sure that they don’t order anything they are allergic to.

The implementation: The backend of the project was built using C# and ASP.NET connected to the MS SQL Server hosted in Azure. Bootstrap was used to build the frontend.
