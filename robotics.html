School Projects

Date: September 2018

Block Avoidance: The robot follows a black line on the table and avoids any Lego blocks on the way by making a curve turn.

Curling: Two robots compete to dominate a spot within a given time limit. The robot that is closest to the spot, wins.

Search and Rescue: The robot manoeuvres through a map scanning a series of barcode patterns to find the right barcode (i.e. prisoner) and then return to a finish spot.

Side Group Projects

Date: December 2019

Maze Search:The robot manoeuvres through a maze made using Lego blocks to find the quickest way out of the maze. It was done mostly in a simulator, but we could not complete the project due to some technical difficulties. Our team implemented algorithms like depth-first search and breadth-first-search and came up with different solutions. However, we were still struggling to store the duration of movements of the robot due to the limitation of the Aseba programming language with the robot used. A stopwatch was built to keep track of the duration of the movement, to optimize the program. The GitHub link shows the stopwatch we implemented for the project.
