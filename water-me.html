Water Me

A mobile app that notifies you, when your plant needs water

Start Date: June 2020; Project State: Prototype-1 under test; Project Type: Personal

The idea: An app that lets you know if your plant needs water through push notifications. The interface includes the current state of the water moisture level. However, to make it more fun, and animated plant will be present in the UI that would talk to the user with a small dialogue box above it where it would say various things like "I am thirsty. Need water!" or "I'm full. Will vomit now!" and a lot more!

Motivation behind the project: I was told to water the plants while my aunt was off in vacation. I forgot to water them regularly. Eventually some of her indoor plants died. So I thought wouldnâ€™t it be a nice idea if my phone reminded me if the plants needed water. If they got too much or too little.

The implementation: The Raspberry Pi will be connected to a soil moisture sensor and send real-time data to a live database. An Android app will collect those data from the database and present it on the display. Python code was used to write the program for the Raspberry Pi and Android Studio was used to build the app. Firebase was used as the database.

Current state: The hardware setup is complete. Housing was made using a tiffin box which has the raspberry pi, the breadboard glue inside it. Holes have been drilled for the external cables. As for the app, it is able to tell how much soil moisture there is. Right now the whole app and setup are being tested on a real plant.
